# Loan calculator

Loan calculator that sends amount and duration to backend.\
Filters out lenders depending on their rules and returning an array with all the lenders that has received the application.

## Available scripts client

In the client directory, you can run:

### `npm install`

To install all dependencies.

### `npm start`

To start the app on port :3000

## Available scripts backend

In the backend directory, you can run:

### `npm install`

To install all dependencies.

### `npm dev`

To start the backend with nodemon on port :5001.

### `npm start`

To start the backend without nodemon on port :5001.
